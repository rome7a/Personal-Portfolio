Work

Educational Chat Bot With Web Interface

For my master's project, I developed an "Educational Chat-bot: Web-interface with an Artificial Intelligence Model" to help educators manage the overwhelming number of student queries. The chat-bot, powered by a BERT model, provided timely, accurate responses to students' questions, particularly aiding those in the Data Structures course at the University of Georgia. Using React JS for the front-end and MySQL for the back-end, the project transitioned from XAMPP to Docker for better scalability and integration. An administrative interface allowed instructors to manage the chat-bot's knowledge base and user access, significantly reducing their workload. This project highlights the potential of AI to enhance education by making it more accessible and responsive to student needs. .

Optimizing Feature Extraction Models for Celebrity Image Recognition: A Comparative Analysis Using OpenCV

In our completed project, we conducted a thorough comparative analysis of Feature Extraction (FE) models for object detection using datasets including CelebFaces Attributes (CelebA) and others from platforms like Kaggle and UCI Machine Learning Repository. Utilizing OpenCV, we focused on evaluating the vantage point tree and histogram models. After rigorous data cleaning and preparation, the vantage point tree model effectively identified images with relevant point features, which were then accurately labeled using the histogram model. Through meticulous model construction, training, and tuning, we achieved optimized performance metrics in terms of efficiency and accuracy. This project not only enhances our understanding of FE models in computer vision but also underscores their practical application in improving image recognition systems, particularly for identifying celebrities in diverse datasets.
